HTML定义了网页的内容
CSS描述了网页的布局
JavaScript网页的行为，是一种轻量级的脚本语言

主要内容



  • 直接写入HTML输出流

  • 对事件的反应

  • 改变HTML的内容

  • 改变HTML的图像

  • 改变HTML的样式

  • 验证输入


JavaScript 用法


必须在script标签中（src属性可以用来使用外部.js文件，外部.js文件不能使用script标签），通常把函数放在head或底部

JavaScript 输出



  • 使用windows.alert弹出警告框

  • 使用document.write()方法将内容写到HTML文档中

  • 使用innerHTML写入到HTML元素

  • 使用consule.log()写到浏览器的控制台


JavaScript 语法



  • 数组：[1,2,3] var a=new array()

  • 对象：{name:”Bob”,age:20}

  • 函数：function myFunction(a,b){return a*b;}

  • 变量：var


JavaScript 语句


用分号来做结束，会忽略多余的空格

JavaScript 变量


undefined表示变量不含值，null用来清空变量，var v=new Number;声明变量类型

JavaScript 事件



  • onchange:HTML元素改变

  • onclick


JavaScript 类型转换


Number()转化为数字、String()转化为字符串、Boolean()转化为布尔值

constructor属性返回所有变量的构造函数

null用于对象、undefined用于变量、属性和方法

JavaScript 正则表达式

CSS:控制多重网页的样式和布局

CSS链接样式


CSS链接样式有顺序要求：

a:link {color:#000000;} 未访问链接

a:visited {color:#00FF00;} 已访问链接

a:hover {color:#FF00FF;} 鼠标移动到链接上

a:active {color:#0000FF;} 鼠标点击时

颜色（字或背景）、字体大小等样式可以改变

CSS列表


可以设置ol、ul前面的样式list-style-type属性

CSS表格

CSS盒子模型


http://www.runoob.com/images/box-model.gif

CSS边框(Border)


CSS轮廓(Outline)


是绘制于元素周围的一条线，位于边框的外围，可以起到突出元素的作用

CSS margin(外边距)padding(内边距)


CSS Display(显示)与Visibility(可见性)


display属性设置一个元素应该如何显示，Visibility属性设置一个元素是显示还是隐藏

CSS Position


Position属性指定了元素的定位类型：


  • static:默认值（即没有定位）

  • fixed:元素的位置相对于浏览器窗口是固定位置，即使窗口是滚动的它也不会移动

  • relative:相对其正常位置

  • absolute:相对于最近的已定位父元素，如果元素没有已定位的父元素，那么它的位置相对于html标签

  • *sticky:基于用户的滚动位置来定位


重叠的元素：z-index属性指定了一个元素的堆叠顺序
