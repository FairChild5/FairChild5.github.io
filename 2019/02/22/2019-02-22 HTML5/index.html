    HTML5 Web存储


    一个比cookie更好的本地存储方式

    客户端存储数据:

    • localStorage:用于长久保存整个网站的数据，没有时间限制

    • sessionStorage:临时保存窗口的数据，在关闭窗口时删除数据


      • 例如：记录一个网站的浏览次数

      HTML5 WebSQL数据库


      三个核心方法：

      1. openDatabase：这个方法使用现有的数据库或者新建的数据库创建一个数据库对象。

      2. transaction：这个方法让我们能够控制一个事务，以及基于这种情况执行提交或者回滚。

      3. executeSql：这个方法用于执行实际的 SQL 查询。


        4. http://www.runoob.com/html/html5-web-sql.html
