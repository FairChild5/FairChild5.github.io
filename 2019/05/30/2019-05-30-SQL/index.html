2019-05-30 SQL

明天要数据库上机，继续复习SQL🐟句

一定要看清题目要求啊！！不要忘记改列名和排序！！

top语句


SELECT TOP number|percent column_name(s)
FROM table_name

limit语句


select from Customer LIMIT 10 ;–检索前10行数据，显示1-10条数据

select from Customer limit 5,10 ;–检索从第6行开始向前加10条数据，共显示id为6,7….15

SELECT * FROM table LIMIT 4,-1;从第5个数据查询，到最后一个数据

case when


case..when..then..else..end

🌰case catname when ‘计算机’ then 3price else 2*price end

计数distinct的列


使用group by

找最大的几条记录


内层：top 1 外层循环：看一下谁等于top 1

日期📅


DATEADD() 在日期中添加或减去指定的时间间隔

DATEDIFF() 返回两个日期之间的时间

SQL 函数


first():SELECT FIRST(OrderPrice) AS FirstOrderPrice FROM Orders 结果返回”OrderPrice” 列的第一个值

ucase() lcase():把字段转化成大、小写

mid():SELECT MID(City,1,3) as SmallCity FROM Persons
结果：”City” 列中提取前 3 个字符

round():SELECT ProductName, ROUND(UnitPrice,0) as UnitPrice FROM Products 结果：把名称和价格舍入为最接近的整数
