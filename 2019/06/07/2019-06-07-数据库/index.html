ER Model


多对一关系：at most 1 箭头指向的一端是“唯一的”

圆箭头：有且只有一个

多端关系转换成双端：关系变实体，注意多对一的箭头

关系中的角色：只需要写一个实体

子类：1、三角形的尖指向父类 2、每种实体都是包含根节点的一棵子树 3、只有根有主键，并且充当所有实体的主键

模型约束：实体完整性（若属性A是基本关系R的主属性，则属性A不能取空值）、参照完整性（外键或者取参照关系中的某个主键值，或者取空值（表示还没有分配）
，外键可以由多个属性构成）、域约束、用户自定义约束

弱实体集合：双边矩形 多对一的sopporting relationship：双边菱形

The Relational Data Model


把子类转换成关系模式

OO方法：每个对象只属于一张表（按照子树转换）

ER方法：一个矩形转化为一张表，子类要带上主键

NULL方法：把所有属性都放到一张表里
