出成绩提醒插件

昨天老师演示了一下简单的谷歌插件，最近等成绩刷新的心烦，想写一个离散成绩出之后给我发消息的插件。遇到了很多问题，而且最后也没做出来😐但是还是要总结一下

总体思路

Python爬虫模拟登录eamis -> 在网页中插入插件（静态）-> 服务器运行Python脚本

爬虫

参考网站：

https://baijiahao.baidu.com/s?id=1612319421218751139&wfr=spider&for=pc

https://www.cnblogs.com/zhaof/p/7284312.html

插件

参考网站：https://github.com/sxei/chrome-plugin-demo

主要是写js文件(jQuery)

服务器

终端登录服务器：ssh root@148.70.181.82

服务器运行Python：需要配置Python环境，然后在终端运行即可

补充

Python中有定时任务框架apscheduler，可以用来定时运行一段代码

参考网站：https://blog.csdn.net/somezz/article/details/83104368
